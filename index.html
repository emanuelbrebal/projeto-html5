O que é a internet? Voltar

A internet, resumidamente falando, é uma rede de computadores de tamanho global. Uma rede de computadores é criada quando dois ou mais computadores se conectam fisicamente (por meio de cabos) ou sem fio (WIFI), afim de trocar informações.

A internet foi criada em 1969, quando o governo americano precisou trocar dados militares secretos entre suas bases que ficavam de um canto a outro em seu continente. Garantindo que a informação fosse enviada de uma forma mais segura e rápida. A rede foi criada pelo orgão de pesquisa americano ARPA e logo ficou conhecida como ARPANET. Logo depois, foi instalada na rede de defesa americana, trocando de nome para DARPANET.

A primeira mensagem da internet.Voltar

A primeira mensagem enviada pela rede ARPA foi na verdade no primeiro dia de conexão. Quando os pesquisadores da ARPANET resolveram enviar uma mensagem contendo o texto "LOGIN" e o outro lado acabou recebendo apenas a mensagem "LO". Mesmo sendo o dia em que a primeira conexão foi feita, também foi a primeira vez que a conexão ficou fora do ar.

Desenvolvimento Web

Linguagens de Programação voltadas ao Web Development Voltar

Com o surgimento da internet também ouve a necessidade de padronizar o desenvolvimento de suas primeiras páginas. Com a aparição dos protocolos da WEB também surgiram as linguagens de programação, sendo a tríade mais conhecida: HTML, CSS e JavaScript. As linguagens de programação voltadas ao desenvolvimento WEB são divididas em Client side e Server side, que serão explicadas mais a frente.

Client Side Voltar

Client side significa: lado do cliente. A saber, as linguagens client side são aquelas que "rodam" no computador do cliente (usuário). Numa aplicação WEB, o computador do cliente faz a requisição ao servidor que, em caso de sucesso, protamente envia os arquivos ao navegador, que os interpreta na máquina do cliente, formando as páginas WEB. Também são conhecidas pelo termo Front End. Exemplos de linguagens web client side são: HTML, CSS, JavaScript e entre outras.

Server Side Voltar

Já as linguagens Server side seguem o mesmo princípio. São linguagens que "rodam" no lado do servidor. O servidor são computadores mais potentes que tem a função de servir algum serviço (chega a ser algo redundante, mas é isso mesmo). Serviços estes que seguem um modelo de arquitetura de redes. Exemplos de linguagens server side: PHP, ASP, Java, Ruby, Python e entre outras.

HTML5

O que é? Voltar

HTML5 é a quinta versão da HyperText Markup Language, ou linguagem de marcação de Hipertexto. A saber, html não é uma linguagem de programação de fato. Ela apenas edita e formata o hipertexto, imagens e outros elementos de uma página web.

Tags básicas Voltar

    A carinha de um arquivo HTML

Todo arquivo html se inicia com a declaração "!DOCTYPE html" dentro das "< >", como todas as tags HTML. Logo em seguida, inicia-se uma tag "html" que será a limitação das linhas de código de seu arquivo, todo o software deve ser escrito dentro dessa tag. E então, uma tag "head" e "body". Toda a parte do "head", como o próprio nome sugere, diz respeito a "cabeça" ou ao "cabeçalho" do site, esta parte é uma área mais de configurações e geralmente não tem muitos elementos que aparecem para o usuário final. Já o "body" é a parte que o usuário interage diretamente com a página WEB. É lá que está presente todo o texto e a formatação básica do site, todo o conteúdo que o usuário final pode interagir.

As tags básicas de formatação Voltar

como exemplos de tags principais nós temos as tags "h". Ela tem 6 tamanhos (indo da h1 até a h6). A tag "h" vem de "header", que significa cabeçalho, e formata em tamanhos diferentes justamente os cabeçalhos da página. Como por exemplo:

h1 para mais ênfase

h2 ainda ênfase

h3 um pouco menos

h4 quase nenhuma

h5 sem ênfase
h6 ainda consegue ler?

elas servem para diferenciar os diferentes parágrafos de conteúdo da página, em grau hierarquizado de tamanhos.

falando em parágrafos, a tag "p" designa justamente esse bloco de texto. Ela está presente em quase todo o site. E esse texto foi escrito usando a tag "p".

Ainda temos algumas tags de formatação de texto como por exemplo:
"strong"
"small"
"i"
"mark"
e muitas outras tags. Lembrando que a maioria das tags tem a formatação "tag> /tag>", ou seja, devemos fechar a maioria das tags que utilizarmos.

Para estudar mais tags HTML é recomendado visitar esse site: https://www.w3schools.com

Atributos Voltar

Como as tags já foram explicadas, também é importante ter noção sobre os "Atributos". Eles servem para formatar ainda mais as tags e deixá-las mais próximas do que queremos. Algumas tags tem atributos necessários para funcionar como é o caso da tag "a", anchor.
Para a tag "a" funcionar, é necessário definir o local da ancoragem. Esse endereço é definido no atributo "href". Outro exemplo comum é a tag "img", que por si só não faz nada, mas com o atributo "src" permite adicionar uma imagem à página. E caso a imagem seja naturalmente muito grande, ainda é possível definir o "width" (a largura) da imagem em pixels. Segue exemplo:

Imagem do planeta terra no tamanho nativo.

Imagem do planeta terra no tamanho de 200 pixels de largura.

É possível verificar que com apenas uma mudança de atributos, o elemento do site se torna mais agradável e de melhor visibilidade.
